Código-Fonte

O comando document.getElementById('teste') é escrito em linguagem JavaScript.

Para melhor vizualização do termo, podemos utilizar a tag "code". Ex:document.getElementById('teste')

Vejamos agora um exemplo de uma fórmula escrita na linguagem Javascript onde utilizamos o "code" e o comando "pre", que fideliza a forma como foi escrita:

        
            num = int(input('digite um número'))
            if num % 2 == 0:
                print(f'O número {num} é par')
            else:
                print(f'O número {num} é ímpar')
            print('fim do programa')

caso contrário seu código ficaria apresentado no site desta maneira:

num = int(input('digite um número')) if num % 2 == 0: print(f'O número {num} é par') else: print(f'O número {num} é ímpar') print('fim do programa')

sem quebras de linhas, ou seja, sem a formatação desejada.
